jlinops
==========================================

`jlinops` is a simple library for implementing "black-box" linear operators accessible via matrix-vector product queries. This library is built on-top of 

The library features:

* Operators for representing matrix factorizations, inverses, and pseudoinverses
* Conversion to/from GPU array compatible linear operators via CuPy
